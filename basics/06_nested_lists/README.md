# Nested Lists in Python (Beginner Friendly)

A **nested list** is a list inside another list.

Example:

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
```

You can think of it like rows and columns.

---

## What to learn first

1. Access values using two indexes: `list[row][col]`
2. Loop through rows
3. Loop through every element using nested loops
4. Add/update values

## Quick memory tip

- Outer list = container of rows
- Inner list = one row
- `data[i][j]` = row `i`, column `j`
